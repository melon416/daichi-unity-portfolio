## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/daichi/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Personal Information

Update the following files to customize your information:

1. **Hero Section** (`components/Hero.tsx`):
   - Change name, title, and description
   - Update avatar image
   - Modify call-to-action buttons

2. **About Section** (`components/About.tsx`):
   - Update personal description
   - Modify statistics
   - Change about image

3. **Contact Information** (`components/Contact.tsx` and `components/Footer.tsx`):
   - Update email, phone, and location
   - Modify social media links

4. **Experience** (`components/Experience.tsx`):
   - Update work experience
   - Modify education details
   - Add/remove certifications

5. **Projects** (`components/Projects.tsx`):
   - Add your projects
   - Update project descriptions and technologies
   - Modify project links

6. **Skills** (`components/Skills.tsx`):
   - Update skill categories
   - Modify skill levels
   - Add/remove technologies

### Styling

The website uses Tailwind CSS for styling. You can customize:

- Colors in `tailwind.config.js`
- Global styles in `app/globals.css`
- Component-specific styles in individual component files

### Images

Replace placeholder images with your own:

- Avatar image in Hero section
- About section image
- Project screenshots
- Resume PDF
